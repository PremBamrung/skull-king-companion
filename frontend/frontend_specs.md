

### **II. Color Palette (CSS Variables)**

Use these hex codes to replicate the rulebook's warm, aged tones.

| Variable Name | Hex Code | Usage |
| --- | --- | --- |
| **`--bg-wood`** | `#4A3B2A` | The "table" background behind the game board (dark oak). |
| **`--bg-parchment`** | `#E6DBC4` | The main game area background (aged paper). |
| **`--text-ink`** | `#2B221B` | Primary text color (soft, warm black). |
| **`--accent-red`** | `#A83226` | "Journal de bord" alerts, error messages, delete buttons. |
| **`--highlight-gold`** | `#D4AF37` | Winning effects, "Bid" success, current player highlight. |
| **`--suit-green`** | `#4E7A40` | Parrot suit accents. |
| **`--suit-purple`** | `#5D4078` | Treasure Map suit accents. |
| **`--suit-yellow`** | `#D9A42C` | Chest suit accents. |
| **`--suit-black`** | `#1A1A1A` | Jolly Roger suit accents. |
